<div id="overview">
  <h2>Executive Summary</h2>
  <p>

It acts as the <strong>Executive Brain</strong> for your personal and professional digital life, managing complex, multi-agent workflows and autonomous decision-making while ensuring you retain 100% sovereignty over your data and hardware.
  </p>

  </p>

<div id="quick-start">
  <h2>Quick Start (TL;DR)</h2>
  <p><strong>Runtime:</strong> Python ≥3.10</p>

  <h4>1. Install the core framework</h4>
  <pre><code>pip install polytope-executive</code></pre>

  <h4>2. Launch the Gateway & Initialize Vaults</h4>
  <p>The wizard creates unique Simplicial Vaults and links hardware biometrics.</p>
  <pre><code>polytope onboard --install-daemon</code></pre>

  <h4>3. Execute an Objective</h4>
  <pre><code>polytope execute "Research the Q1 market trends and draft a report in my G-Drive"</code></pre>
</div>

<div id="architecture">
  <h2>Simplicial Architecture</h2>
  <p>Unlike flat assistants, Polytope operates on a Hierarchical Orchestration model:</p>
  <pre>
          [ User Objective ]
                  │
                  ▼
┌──────────────────────────┐      ┌──────────────────────────┐
│    EXECUTIVE AGENT       │&lt;────&gt;│    VERUS_IDENTITY        │
│   (Strategic Planner)    │      │  (Cryptographic Proof)   │
└─────────┬────────────────┘      └──────────────────────────┘
          │
          ├─► [ Simplicial Vault A ] ──► iMessage / WhatsApp
          ├─► [ Simplicial Vault B ] ──► G-Drive / Slack
          ├─► [ Simplicial Vault C ] ──► Local Python Sandbox
          └─► [ Evaluation Loop ]    ──► (Critic / Validation)
  </pre>
</div>

<div id="subsystems">
  <h2>Core Subsystems</h2>
  
  <div>
    <h3>1. Security & Trust Protocol</h3>
    <ul>
      <li><code>ONE_TOUCH_LOGIN</code>: Biometric-tethered execution (FaceID/TouchID).</li>
      <li><code>SIMPLICIAL_ISOLATION</code>: Cryptographically isolated "Vaults" for each bridge.</li>
      <li><code>AUTONOMY_LEVEL</code>: Toggle between Restricted, Semi-Autonomous, and Full Sovereign.</li>
    </ul>
  </div>

  <div>
    <h3>2. The Social Manifold</h3>
    <ul>
      <li><strong>Messaging:</strong> WhatsApp (QR_Sync), Telegram, Signal (E2EE), iMessage (Secure Tunnel).</li>
      <li><strong>Social:</strong> Instagram, Facebook, X (OAuth2).</li>
    </ul>
  </div>

  <div>
    <h3>3. Identity & Proof</h3>
    <ul>
      <li><strong>VerusID Integration:</strong> Every action is cryptographically signed, providing a "Digital Passport" and "Proof of Origin" via <code>VERUS_IDENTITY</code>.</li>
    </ul>
  </div>
